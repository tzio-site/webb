JavaScript Tutorial

JS is the brain of a website, used to make HTML and CSS interactive and dynamic. It uses features like functions, events, and DOM manipulation to respond to user actions and update the page in real time.


let / const

Purpose: Variables store information that you can use and change later.
Difference: let → value can change
const → value cannot change after setting

let name = "Ali"; name = "Omar"; // ✅ allowed const PI = 3.14; PI = 3.15; // ❌ error

alert()

Purpose: Shows a small browser alert box with your message.
When to use: For quick notices, but not good for serious UI because it blocks the page until closed.

alert("Welcome to my website!");

console.log()

Purpose: Displays messages in the browser console (press F12 → Console tab).
Why use it: Helps you check if your code is working or see values while testing.

console.log("Page loaded");

document.getElementById()

Purpose: Lets you access an element in HTML so you can change it.
Use with: .innerHTML to change text, .style to change CSS, etc..

document.getElementById("title").innerHTML = "New Title";

addEventListener()

Purpose: Makes your site respond when something happens (click, hover, typing, etc.).
Why it’s better than HTML onclick: Keeps HTML and JS separate.

document.getElementById("btn").addEventListener("click", function() { alert("Button clicked!"); });

if...else Tag

Purpose: Runs different code depending on a condition (true/false).
Example: Check age to allow entry.

let age = 16; if (age >= 18) { console.log("Allowed"); } else { console.log("Not allowed"); }

for loop

Purpose: Runs code multiple times without writing it repeatedly.

for (let i = 1; i <= 3; i++) { console.log("Number: " + i); }

function

Purpose: Groups code together so you can call it anytime without rewriting.

function greet(user) { console.log("Hello, " + user); } greet("Sara"); greet("Omar");
