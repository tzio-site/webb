React Tutorial

React provides a component-based approach for building user interfaces. Below are some essential React concepts with examples:

Components (Function Components)

What it is: The building blocks of React apps. A component is just a JavaScript function that returns JSX.

function Welcome() { return <h1>Hello, React!</h1>; }

JSX (JavaScript XML)

What it is: A syntax that looks like HTML but works inside JavaScript, allowing you to write UI elements.

const element = <p>This is JSX </p>;

Props (Properties)

What it is: A way to pass data from a parent component to a child component.

function Greeting({ name }) { return <h1>Hello, {name}!>/h1>; } // Usage <Greeting name="Hou" />

State

What it is: React’s way to store data inside a component that can change over time.

import { useState } from "react"; function Counter() { const [count, setCount] = useState(0); return ( <div> <p>Count: {count}</p> <button onClick={() => setCount(count + 1)}>Increase</button> </div> ); }

Event Handling

What it is: React handles events like onClick similar to HTML but uses camelCase.

function Button() { function handleClick() { alert("Button clicked!"); } return <button onClick={handleClick}>Click Me</button>; }

Conditional Rendering

What it is: Show elements based on a condition.

function LoginStatus({ isLoggedIn }) { return <p>{isLoggedIn ? "Welcome back!" : "Please log in."}</p>; }

Lists and Keys

What it is: Render multiple elements using .map() and give each a unique key.

function TodoList() { const todos = ["Learn React", "Build Projects", "Get a Job"]; return ( <ul> {todos.map((todo, index) => ( <li key={index}>{todo}</li> ))} </ul> ); }

useEffect Hook

What it is: Run code when a component renders or when certain data changes.

import { useState, useEffect } from "react"; function Timer() { const [seconds, setSeconds] = useState(0); useEffect(() => { const interval = setInterval(() => setSeconds(s => s + 1), 1000); return () => clearInterval(interval); }, []); return <p>Time: {seconds}s</p>; }

Styling in React

What it is: Add CSS directly or use style objects.

function StyledText() { const style = { color: "blue", fontSize: "20px" }; return <p style={style}>Styled with inline CSS</p>; }

Component Composition

What it is: Combine components together.

<audio controls> function Header() { return <h1>My Website</h1>; } function App() { return ( <div> <Header /> <p>Welcome to my site!</p> </div> ); }
