HTML Tags Tutorial

HTML provides the structural foundation for web pages. Below are some essential HTML tags with examples:

<header> Tag

The <header> element represents introductory content, typically containing navigation links, logos, or heading elements. It's commonly used at the top of a webpage or section.

<header> <h1>Website Title</h1> <nav> <ul> <li><a href="#home">Home</a></li> <li><a href="#about">About</a></li> </ul> </nav> </header>

<link> Tag

The <link> element specifies relationships between the current document and external resources. It's most commonly used to link CSS stylesheets, but can also link icons, alternative versions of the page, and more.

<!-- Link to a CSS stylesheet --> <link rel="stylesheet" href="styles.css"> <!-- Link to a favicon --> <link rel="icon" type="image/png" href="favicon.png"> <!-- Link to a preload resource --> <link rel="preload" href="font.woff2" as="font" type="font/woff2" crossorigin>

<div> Tag

The <div> is a generic container for flow content. It has no effect on the content or layout until styled with CSS. Divs are commonly used to group elements for styling purposes or as hook points for JavaScript.

<div class="card-container"> <div class="card"> <h3>Card Title</h3> <p>This is a card component created using div elements.</p> </div> <div class="card"> <h3>Another Card</h3> <p>Divs help organize content into logical sections.</p> </div> </div>

<section> Tag

The <section> element represents a thematic grouping of content, typically with a heading. It should make sense on its own when separated from the rest of the page.

<section> <h2>About Us</h2> <p>We are a company dedicated to web development.</p> </section> <section> <h2>Our Services</h2> <ul> <li>Web Design</li> <li>Frontend Development</li> </ul> </section>

<article> Tag

The <article> element represents a self-contained composition that is independently distributable or reusable (e.g., blog post, news story).

<article> <h2>How to Learn HTML</h2> <p>HTML is the foundation of web development...</p> <footer> <p>Posted on <time datetime="2023-05-15">May 15</time> by Author</p> </footer> </article>

<nav> Tag

The <nav> element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.

<nav> <ul> <li><a href="/home">Home</a></li> <li><a href="/about">About</a></li> <li><a href="/contact">Contact</a></li> </ul> </nav>

<form> Tag

The <form> element represents a document section containing interactive controls for submitting information to a web server.

<form action="/submit" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name" required> <label for="email">Email:</label> <input type="email" id="email" name="email" required> <label for="message">Message:</label> <textarea id="message" name="message"></textarea> <button type="submit">Send</button> </form>

<table> Tag

The <table> element represents tabular data — information presented in a two-dimensional table comprised of rows and columns of cells containing data.

<table> <thead> <tr> <th>Name</th> <th>Age</th> <th>Occupation</th> </tr> </thead> <tbody> <tr> <td>John Doe</td> <td>28</td> <td>Web Developer</td> </tr> <tr> <td>Jane Smith</td> <td>32</td> <td>UX Designer</td> </tr> </tbody> </table>

<video> Tag

The <video> element embeds a media player which supports video playback into the document. You can specify multiple sources for browser compatibility.

<video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> <source src="movie.ogg" type="video/ogg"> Your browser does not support the video tag. </video>

<audio> Tag

The <audio> element embeds sound content in documents. It may contain one or more audio sources, represented using the src attribute or the <source> element.

<audio controls> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> Your browser does not support the audio element. </audio>

<picture> Tag

The <picture> element contains zero or more <source> elements and one <img> element to offer alternative versions of an image for different display/device scenarios.

<picture> <source media="(min-width: 1200px)" srcset="large.jpg"> <source media="(min-width: 768px)" srcset="medium.jpg"> <img src="small.jpg" alt="Description"> </picture>

<p> Tag

The <p> element defines a paragraph. Browsers automatically add some margin before and after each paragraph.

<p>This is a paragraph.</p> <p>This is another paragraph.</p>

<a> Tag

The <a> element creates hyperlinks to other web pages, files, locations within the same page, or any other URL.

<a href="https://example.com">Visit Example.com</a> <br> <a href="about.html">About Us</a> <br> <a href="#top">Back to Top</a>

<img> Tag

The <img> element embeds an image into the document. Always include the alt attribute for accessibility.

<img src="logo.png" alt="Company Logo" width="200"> <br> <img src="photo.jpg" alt="Beautiful landscape" style="width:100%">

<ul> and <ol> Tags

<ul> creates an unordered (bulleted) list, while <ol> creates an ordered (numbered) list. Both use <li> for list items.

<!-- Unordered List --> <ul> <li>First item</li> <li>Second item</li> </ul> <!-- Ordered List --> <ol> <li>Step one</li> <li>Step two</li> </ol>

<button> Tag

The <button> element creates a clickable button that can trigger JavaScript functions or submit forms.

<button onclick="alert('Clicked!')">Click Me</button> <br> <button type="button" disabled>Disabled Button</button> <br> <button style="background-color: #8FB3E2; color: white;">Styled Button</button>

<span> Tag

The <span> element is an inline container used to mark up part of a text or document. It's often used with CSS or JavaScript.

<p>This is <span style="color:red">red text</span> inside a paragraph.</p> <p>The word <span class="highlight">important</span> is highlighted.</p>

<br> Tag

The <br> element inserts a single line break. It's an empty tag (no closing tag needed).

This text is before the line break.<br> This text is after the line break.
