Css Tags Tutorial

CSS is the skin of a website, used to make HTML look attractive and organized. It uses properties like color, font-size, and margin to control how elements appear on the page. While HTML builds the structure, CSS brings it to life visually with colors, layouts, and designs.


.color Tag

The color property in CSS is used to change the text color of an element. It can take values like color names (red, blue), hexadecimal codes (#ff0000), RGB values (rgb(255,0,0)), or even HSL values (hsl(0, 100%, 50%)).

p { color: blue; /* This makes paragraph text blue */ }

.font-size Tag

The .font-size property sets how large or small the text appears. You can use units like pixels (px), relative sizes (em, rem), or percentages (%).

p { font-size: 20px; /* Text will be 20 pixels tall */ }

font-family Tag

The font-family property changes the typeface of the text. You can use common fonts like "Arial", "Times New Roman", or web fonts from Google Fonts.

p { font-family: Arial, sans-serif; /* Arial text with sans-serif fallback */ }

text-align Tag

The text-align property sets the horizontal alignment of text inside its container. Common values: left, center, right, and justify.

p { text-align: center; /* Centers the text */ }

font-weight Tag

The font-weight property controls how thick or bold the text is. Common values: normal, bold, or numbers like 100 to 900.

p { font-weight: bold; /* Makes text bold */ }

background-color Tag

The background-color property sSets the background color of an element.

div { background-color: lightgray; }

margin Tag

The margin Creates space outside the element’s border.

div { margin: 20px; }

padding Tag

The padding Creates space inside the element’s border (around content)

div { padding: 15px; }
